For Developers: Technical Documentation

The LGWD AI Chatbot plugin is built with modern WordPress standards and provides extensive customization options for developers.

Installation Requirements

  • WordPress 5.0 or higher
  • PHP 7.4 or higher
  • Active n8n instance with Chat Trigger capability
  • SSL certificate (recommended for secure API communication)
  • Modern web browser with JavaScript enabled

Basic Installation Steps

  1. Download Plugin: Get the latest version from our GitHub repository
  2. Upload to WordPress: Admin > Plugins > Add New > Upload Plugin
  3. Activate Plugin: Enable the LGWD AI Chatbot in your plugins list
  4. Configure n8n Workflow: Create Chat Trigger node with embedded mode
  5. Set CORS Policy: Add your domain to allowed origins in n8n
  6. Enter Chat URL: Add the n8n chat endpoint in WordPress admin settings
  7. Customize Appearance: Set colors, positioning, and messages
  8. Test Functionality: Verify chat integration works correctly

n8n Workflow Configuration

Required Nodes:

  • Chat Trigger: Set to "embedded" mode with your domain in CORS settings
  • AI Agent: Configure with OpenAI, Claude, or your preferred AI service
  • Memory Node: Maintains conversation context across messages

Optional Enhancements:

  • Email nodes for lead notifications and follow-up
  • CRM nodes for automatic contact creation
  • Database nodes for conversation logging
  • Webhook nodes for third-party integrations
