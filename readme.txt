== Installation ==

= Automatic Installation =

1. Go to WordPress Admin → Plugins → Add New
2. Search for "LGWD AI Chatbot"
3. Click Install Now and then Activate
4. Go to AI Chatbot in your admin menu to configure

= Manual Installation =

1. Download the plugin zip file
2. Go to WordPress Admin → Plugins → Add New → Upload Plugin
3. Upload the zip file and click Install Now
4. Activate the plugin
5. Configure through AI Chatbot settings

= n8n Setup Requirements =

1. Create a Chat Trigger node in n8n (set to "embedded" mode)
2. Add your domain to CORS settings in the Chat Trigger
3. Connect an AI Agent node (OpenAI, Claude, etc.)
4. Add a Memory node for conversation context
5. Copy the Chat URL (ends with /chat)
6. Enter the URL in your WordPress plugin settings

== Frequently Asked Questions ==

= Do I need an n8n account? =

Yes, this plugin requires an active n8n instance with Chat Trigger capability. You can use n8n Cloud or self-hosted n8n.

= What AI providers are supported? =

The plugin works with any AI provider supported by n8n, including OpenAI GPT-4, Claude, local models, and more.

= Is it mobile responsive? =

Yes, the chatbot is fully optimized for mobile devices with responsive design and touch-friendly interactions.

= Can I customize the appearance? =

Absolutely! You can customize colors, positioning, messaging, greeting text, and more through the WordPress admin interface.

= Does it capture leads? =

Yes, when properly configured with n8n workflows, the AI can intelligently capture and qualify leads.

= Do you offer setup services? =

Yes, [Lions Gate Web Design](https://lionsgatewebdesign.com) provides professional n8n workflow setup and plugin customization services.

= Is it GDPR compliant? =

The plugin itself doesn't store personal data. GDPR compliance depends on your n8n workflow configuration and data handling practices.

= Can I use it on multiple sites? =

Yes, the plugin can be installed on multiple WordPress sites. Each site requires its own n8n Chat Trigger configuration.
